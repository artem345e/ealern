# EduLearn — Платформа дистанційного навчання

Вебплатформа для дистанційного навчання на Python/Flask з SQLite.

## Технічний стек
- **Python 3.x** + **Flask** (мікрофреймворк)
- **SQLite** (база даних у файлі)
- **Bootstrap 5** (інтерфейс)

## Структура проєкту
```
elearn/
├── app.py              ← Головний файл (всі маршрути та логіка)
├── elearn.db           ← База даних (створюється автоматично)
├── requirements.txt    ← Залежності
├── static/
│   └── css/style.css   ← Стилі
└── templates/
    ├── base.html       ← Базовий шаблон (навбар, футер)
    ├── index.html      ← Каталог курсів
    ├── login.html      ← Вхід
    ├── register.html   ← Реєстрація
    ├── course.html     ← Сторінка курсу
    ├── lesson.html     ← Перегляд лекції
    ├── teacher.html    ← Панель викладача
    └── profile.html    ← Профіль користувача
```

## Запуск

### 1. Встановити залежності
```bash
pip install -r requirements.txt
```

### 2. Запустити сайт
```bash
python app.py
```

### 3. Відкрити у браузері
```
http://localhost:5000
```

## Тестові облікові записи

| Роль       | Логін    | Пароль     |
|------------|----------|------------|
| Викладач   | admin    | admin123   |
| Студент    | student1 | student123 |

## Функціонал

### Для студентів
- Реєстрація та вхід
- Перегляд каталогу курсів
- Запис на курс
- Перегляд лекцій (тільки після запису)
- Навігація між уроками
- Особистий кабінет з переліком курсів

### Для викладачів
- Створення нових курсів
- Додавання лекцій до курсів
- Видалення курсів
- Перегляд статистики (кількість студентів, уроків)

## База даних

### Таблиці
- **users** — ID, логін, пароль (SHA-256), роль (student/teacher)
- **courses** — ID, назва, опис, ID автора, дата
- **lessons** — ID, ID курсу, заголовок, текст, порядок
- **enrollments** — ID студента, ID курсу (записи на курси)
